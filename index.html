Computer Science Projects

Video Game Trivia

My first project Senior year was a video game trivia site. Along with two other students we designed an application that pulled data from a video game API such as characters, settings, and items, which we created into a variation of different questions. Check it out here!

Price is Right!

My second project was a solo project for the AP exam. For this application I was able to pull data on various items from a Best Buy API, and used the information such as names and prices to create a higher or lower price guessing game similar to Price is Right. Check it out here!

Jang! Food Delivery

My final project was a group project for a Full Stack Competition. Our group created a food delivery application for our school's cafe. We created a log in system, payment system, confirmation page, and many other key features you would find on DoorDash or Uber Eats. Check it out here!

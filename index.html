Hi, my name is Bryan and I am a 2nd year graduate student at Duke University conducting research in the Labratory for Intelligent Systems Technology in the Department of Civil and Environmental Engineering. I am studying both Civil and Electrical Engineering, namely application of IoT systems and machine learning in civil engineering. I did my undergraduate at Princeton University where I graduated summa cum laude in Civil Engineering (structural focus) with a minor in Computer Science. This website hosts some of my current and past research, projects from graduate school, and some passion projects! Clicking on each bubble will lead you to the corresponding code or publication.

Research

Embedded sensing system for shipboard damage control scenarios

This is a ongoing projection in our lab group, in which I am applying machine learning and computer vision models to human physiological data (heart rate, skin temperature) from naval emergency response training exercises to model response to stressors and predict performance under extreme conditions, and track which room of the ship a participant is currently in.

Data Analytics Machine Learning Python SQL

Design and Application of a Cloud-Supported LTE Wireless Sensor for Remote Monitoring Applications

Designed and deployed an IoT sensor system including an autonomous wireless CO2 flux chamber and groundwater monitoring sensor for large-scale environmental monitoring of peatlands for carbon storage data in peatlands which is sent wirelessly via LTE to an AWS server with Postgres for visualization. This is a current project which will be presented at SPIE Smart Structures and Non-Destructive Evaluation in March 2026.

Embedded Systems Low-level Programming AWS SQL

Methods for Identifying and Modeling Strain Anomalies in Prestressed Concrete Structures

Completed a year-long senior thesis project focused on a data-driven approach to diagnosing anomalies in reinforced concrete structures and predicting their long-term growth by training various machine learning models on strain and temperature data.

Structural Health Monitoring Machine Learning Python

Projects

3D FEM Solver for Structures Under Body and Traction Loads

Application for solving a 3D finite element methods problem for a structure under body and traction loads. Code is also provided to check solution using patch test and method of manufactured solutions. Demonstrated using 3D model of the Duke Chapel tower.

Matlab FEM

Compressed Image Sensing Recovery

An algorithm which restores an image with corrupted/missing pixels, by using LASSO regression and random subsets cross validation to estimate the pixel color for black and white images.

Python Machine Learning

Brain-Computer Interface Movement Decoding

An algorithm using a support vector machine (SVM) with Two-Level cross-validation to determine whether someone using a brain-computer interface (BCI) intends to select a “left” or “right” movement.

Python Machine Learning

Trust-Region Newton-Conjugate-Gradient Optimizer in Deep Learning

Integration of a trust region Newton CG optimizer in PyTorch, and compared results with the standard Adam in approximating elementary functions as well as solution to a Physics-Informed Neural Network.

Python Neural Networks
